# Pinterest AMOLED Dark Mode

A beautiful customizable dark mode for Pinterest.

## Features

- AMOLED dark theme
- Light / Dark toggle
- Accent colors
- Contrast modes
- Image-safe rendering
- Dynamic Pinterest support
- Custom floating settings panel

## Installation

Install Tampermonkey:

https://www.tampermonkey.net/

Then install the script:

[Install Script](https://github.com/thetalhachy/Pinterest-Dark-Mode-Toggle-Pro/raw/refs/heads/main/Pinterest-Dark-Mode-Toggle-Pro.user.js)
